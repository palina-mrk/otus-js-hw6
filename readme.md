# Приложение "Прогноз погоды"

- при открытии страницы пользователь видит погоду (город, температуру и иконку) в своей местности (для получения прогноза погоды используется Open Weather Map API)

- он может ввести имя города в поле ввода и увидеть погоду в выбранном городе 

- введенные города сохраняются у пользователя в браузере, так что он видит последние 10 городов, где он смотрел погоду

- при клике по строчке города в списке он видит погоду в выбранном городе

- кроме информации о погоде в центре страницы показывается карта для введенного адреса (используется Yandex Maps Javscript API)

## структура проекта:

- index.html (код страницы)
  
- js/script.js (скрипт)

- css/styles.css (стили)
